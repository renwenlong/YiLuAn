# iOS 冒烟测试清单

> 目的：在 iOS 端验证核心用户流程可正常运行，不要求覆盖所有功能，仅验证关键路径。
> 环境：Xcode 16+ / iOS 17+ 模拟器或真机 / 后端 Staging 或本地开发环境
> 执行方式：手动测试

---

## 前置条件

- [ ] 后端服务已启动（本地 `uvicorn` 或 Staging 环境）
- [ ] iOS 项目可正常编译并运行
- [ ] 已配置 `AppConfig.swift` 中的 API 地址指向正确后端

## 已知限制

- 微信登录在 iOS 端不可用（非微信环境），使用手机号+验证码登录
- 支付在 mock 模式下为即时成功，不会弹出真实支付界面
- 推送通知需要真机 + 正确配置 APNs
