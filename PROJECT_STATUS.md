## 一、项目概况

医路安是一个医疗陪诊服务平台，连接患者与专业陪诊师，提供：
- 全程陪诊（¥299）
- 半程陪诊（¥199）
- 代办跑腿（¥149）

**技术栈：** FastAPI + PostgreSQL + Redis | 微信小程序 | iOS (SwiftUI)
